# Vennperio Hub

Portal público oficial de comunicados do Vennperio.

O Hub mantém o histórico de atualizações, manutenções, incidentes e avisos de
desenvolvimento. O mesmo conteúdo é disponibilizado como uma página acessível
para jogadores e como um feed JSON consumido pelo jogo.

Comunicados podem conter um resumo curto, destaques para exibição dentro do
jogo e um changelog completo dividido entre adições, alterações, correções e
problemas conhecidos.

O projeto é uma aplicação estática publicada em
[hub.vennperio.site](https://hub.vennperio.site/).
